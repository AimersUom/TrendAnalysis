Extract tweets folder contains the code used to extract tweets from twitter.
Impact prediction folder contains the svm code and the dataset(newdata) used to train the model.

preprocessing.py contains the code used to preprocess tweets.
tf_kmeans.py code is used to identify trending keywords and cluster them(method 01).
LDA.py consist of the code to apply LDA model to the data to obtain trending topics for the particular period.

rawdata folder contains tweets collected per two weeks of each month in year 2017.
